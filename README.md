# Load JS

## Solution

* [Concatenate all scripts into `index.js`](scripts/scripts.js).
* [Load `index.js` in head using `defer`](src/_base/layout.html).

This prevents `defer` race issues in <IE10 and prevents queueing of multiple scripts.
